# App Store submission checklist

## Screenshots (required)

- [ ] iPhone 6.7" (1290×2796) — 6 screenshots  
  - Dashboard with active session  
  - Stats screen with data  
  - Friends / leaderboard  
  - Session complete screen  
  - Profile with achievements  
  - Streak milestone  

- [ ] iPhone 5.5" (1242×2208) — same screens as above  

## Metadata

- [ ] App name: Prodify  
- [ ] Subtitle: Producer session tracker (adjust to final marketing copy)  
- [ ] Keywords: music production, beat making, productivity, studio, sessions  
- [ ] Description (4000 characters max)  
- [ ] What’s New (this version)  
- [ ] Support URL: https://prodify.app/support  
- [ ] Privacy policy URL: https://prodify.app/privacy  

## Build

- [ ] Production build: `eas build --platform ios --profile production`  
- [ ] Test on a physical device  
- [ ] Verify API connects to production (`EXPO_PUBLIC_API_URL`)  
- [ ] Test IAP / subscriptions (if applicable)  
- [ ] Test push notifications  

## Review information

- [ ] Test account credentials for App Review  
- [ ] Demo video (optional)  
- [ ] Notes for reviewer  

## Before submit

- [ ] Age rating questionnaire completed  
- [ ] Privacy manifest completed (Apple requirements)  
- [ ] App icon 1024×1024  
- [ ] Localizations match what you ship (currently English-only in-app; add more if listed on the store)
